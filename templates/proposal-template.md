## 📋 提案级别判断

**一级决议**（需要≥2 个 IDE 赞同）：
- 拼写错误、代码格式调整
- 简单注释添加
- 不改变逻辑的重构

**二级决议**（需要≥5 个 IDE 赞同 + 用户投票）：
- 新增基础功能
- 性能优化
- 批量修改

**三级决议**（需要≥8 个 IDE 赞同 + 用户投票）：
- 核心架构修改
- 主业务流程变更
- 核心功能调整
- 数据安全问题

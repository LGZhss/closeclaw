## 🎯 使用场景

### 场景 1: 第一次提交代码

```
1. 阅读 [重要通知](./IMPORTANT_NOTICE.md)
2. 学习 [提交指南](./PROPOSAL_SUBMISSION_GUIDE.md)
3. 使用 [检查清单](./PROPOSAL_CHECKLIST.md) 逐项检查
4. 复制 [提案模板](../../templates/proposal-template.md)
5. 参考 [示例提案](../../votes/proposal-000-example.md)
```

### 场景 2: 发起新提案

```
1. 阅读 [提交指南](./PROPOSAL_SUBMISSION_GUIDE.md) 的提案级别判断
2. 复制 [提案模板](../../templates/proposal-template.md)
3. 使用 [检查清单](./PROPOSAL_CHECKLIST.md) 确保完整
4. 参考 [当前提案](../../votes/proposal-001-document-optimization.md) 学习格式
```

### 场景 3: 参与投票

```
1. 查看 [投票目录](../../votes/README.md) 了解待投票提案
2. 阅读 [当前提案](../../votes/proposal-001-document-optimization.md)
3. 在提案中填写投票表
4. 参考 [协作规则](../guidelines/COLLABORATION_RULES_v3.md) 了解投票规则
```

### 场景 4: 用户提出修改

```
1. 阅读 [重要通知](./IMPORTANT_NOTICE.md) 的用户提出说明
2. 在提案中标注 `**用户提出**: ✅ 是（直接通过）`
3. 参考 [提交指南](./PROPOSAL_SUBMISSION_GUIDE.md) 了解特殊流程
4. 按照正常流程创建提案和投票
```

## ❓ 常见问题

### Q: 我应该先读哪个文档？

**A**: [重要通知](./IMPORTANT_NOTICE.md)（5 分钟），了解核心要求。

### Q: 如何快速学会提交提案？

**A**: 按照 [提交指南](./PROPOSAL_SUBMISSION_GUIDE.md) 的 10 步流程操作。

### Q: 提交前如何检查？

**A**: 使用 [检查清单](./PROPOSAL_CHECKLIST.md) 逐项检查。

### Q: 用户提出的提案如何处理？

**A**: 在提案中标注"用户提出：✅ 是（直接通过）"，详见 [重要通知](./IMPORTANT_NOTICE.md)。

### Q: 违反规则会怎样？

**A**: 根据 [重要通知](./IMPORTANT_NOTICE.md)，第一次警告，第二次暂停权限，第三次降级。

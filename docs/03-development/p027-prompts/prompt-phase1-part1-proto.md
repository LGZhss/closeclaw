# P027 Phase 1: 跨语言 Protobuf 标准化 (Dart ↔ Go ↔ TS)

## 📌 核心目标
为 CloseClaw Ultra 架构建立统一底层通信契约 `proto/messages.proto`。必须包含支持三层流转的 `trace_id` 分布式追踪字段和任务状态生命周期。

## ⚠️ 强制性纪律 (ANTI-LAZINESS)
作为具备自主规划能力的高阶协作主体，如何设计字段随你决定，但你**必须严格遵守**以下铁律：
1. **[绝不留坑]**：严禁在 `.proto` 或生成的代码中遗留任何形式的 `// TODO` 或 `// 待实现`。你提交的代码必须是完整的、即刻可接入 gRPC 的生产级契约。
2. **[完整生成]**：如果环境允许，你必须自行生成对应的语言存根（Stubs），不要指望人类或其他代理帮你擦屁股。
3. **[兼容审查]**：在编写前必须完整查阅旧版 TS `src/types.ts` 中的 `Task` 结构，确保字段 1:1 映射不会丢失业务数据。

完成开发后，提交至 `proposal/027-phase1-part1` 分支。

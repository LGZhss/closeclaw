## 🎯 快速步骤

### 1️⃣ 阅读统一指南
访问：[docs/03-development/onboarding.md](../03-development/onboarding.md)

### 2️⃣ 准备信息

- 大模型信息（必填）
- 能力声明

### 3️⃣ 登记信息

### 4️⃣ 更新配置并提交
- 更新 [`.subjects.json`](../../.subjects.json)
- 提交到 Git

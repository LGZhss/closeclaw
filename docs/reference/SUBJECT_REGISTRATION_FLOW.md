# 协作主体注册流程 (Subject Registration)

> **版本**: 2.1.0 [ACTIVE]
> **目的**: 指导新参与的 AI 助手如何完成“协作主体”身份备案。

---

## 🎯 注册前准备
1. **读宪法**: [RULES.md](../../RULES.md)
2. **读指引**: [COLLABORATOR_ONBOARDING.md](../contributing/COLLABORATOR_ONBOARDING.md)
3. **查白名单**: `/.subjects.json`

---

## 📝 注册步骤

### 1. 创建身份卡片
所有新进主体必须在 `groups/global/CONTEXT.md` 或新注册文件中提交如下信息：

```markdown
### [协作主体名称] 注册申明
- **内部 ID**: [subject_id，如: traecoding]
- **核心模型**: [如: Claude-3.5-Sonnet]
- **优势能力**: [如: 并发重构, 安全审计]
- **协作指纹**: [SessionID 或自定义指纹]
```

### 2. 参与“第一案”
新注册主体建议先对现有 `votes/` 中的提案发表 **CRITIQUE** (批判性审计)，以此建立声誉。

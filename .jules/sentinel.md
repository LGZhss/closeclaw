## 2024-03-28 - Command injection in legacy execution functions
**Vulnerability:** `execAsync` suffered from command injection as arguments were improperly split by a naive regex and executed.
**Learning:** Legacy execution functions might have rudimentary command string parsing. Changing the signature to an array causes runtime regressions. You must use a robust state-machine parser internal to the function to split the string into `executable` and `args`, and keep `shell: false`. `executeSystemCommand` is designed explicitly to execute shell (`powershell.exe`, `/bin/sh`) for the Agent's tool capabilities and MUST NOT be stripped of its shell wrapping, otherwise it breaks the expected tool behavior.
**Prevention:** Always use a state-machine parser for string command executions internally rather than naively matching regex, or enforce passing an array of arguments from the get-go. Do not remove shell environments from commands explicitly designed to emulate shell interaction.

## 2024-03-28 - Command injection in legacy execution functions
**Vulnerability:** `execAsync` argument tokenization naive regex fails to safely handle advanced quote/escape patterns and empty string parameters.
**Learning:** Legacy execution functions might have rudimentary command string parsing. Changing the signature to an array causes runtime regressions. You must use a robust state-machine parser internal to the function to split the string into `executable` and `args` that also perfectly supports tokenizing explicitly empty strings `""`, and keep `shell: false`. `executeSystemCommand` is designed explicitly to execute shell (`powershell.exe`, `/bin/sh`) for the Agent's tool capabilities and MUST NOT be stripped of its shell wrapping, otherwise it breaks the expected tool behavior.
**Prevention:** Always use a state-machine parser for string command executions internally rather than naively matching regex, or enforce passing an array of arguments from the get-go. Do not remove shell environments from commands explicitly designed to emulate shell interaction. And ensure parsing logic handles empty tokens cleanly.

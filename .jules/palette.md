# CloseClaw 协作准则 - Palette 专项引导

🎨 欢迎，Palette。在进行 UI/UX 优化之前，请严格遵守以下本地协作法律：

## 1. 提案先行 (Proposal First)
在发起 Pull Request 之前，必须在 `votes/` 目录下注册提案。
- **文件名**: `proposal-[编号]-ui-ux-[描述].md`
- **必填信息**: 原 UI 存在的问题（如可访问性、对比度）、你的改进方案、预览说明。
- **状态标记**: `✅ 已通过（用户提出特批）`

## 2. 语言规范 (Language Policy)
- **所有输出必须使用简体中文**。
- 包括：提案内容、PR 标题（如 `🎨 Palette: 优化按钮对比度`）、PR 描述及 Commit 备注。

## 3. 提交备注 (Attribution)
在 PR 和 Commit 结尾标注：
- `Proposal-By`: Google Labs Jules (Palette)
- `Implemented-By`: Google Labs Jules (Palette)
- **链接**: 必须在 PR 中附带指向 `votes/` 下对应提案的链接。

---
**Palette's Note**: 
美感与规范并重。如果没有提案文档，你的 PR 将会被系统拦截。请先“画”好文档。

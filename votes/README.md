## 使用方法

### 发起提案

1. 复制提案模板
2. 填写提案内容
3. 创建 worktree 并开发
4. 推送代码并创建 PR

### 参与投票

1. 阅读提案
2. 审查代码（可选）
3. 填写投票
4. 提交投票

# 投票文档目录

此目录存放所有代码修改提案。

## 使用方法

1. 复制提案模板：`cp ../templates/proposal-template.md proposal-001.md`
2. 填写提案内容
3. 发起投票
4. 统计结果

## 文件命名

- proposal-001.md - 第一个提案
- proposal-002.md - 第二个提案
- ...

## 提案流程

1. **起草**: 基于 templates/proposal-template.md 创建提案
2. **投票**: 所有注册协作主体参与投票
3. **合并**: 投票通过后发起 PR 并合并到 main
4. **归档**: 已完成的提案保留在此目录作为历史记录

# 投票文档目录

> **用途**: 存放所有代码修改提案和投票文档
> **位置**: 项目根目录的 `votes/` 目录
> **状态**: 🟢 已创建
> **最后更新**: 2026-03-14

---

## 📝 使用方法

### 发起提案

1. 复制 `templates/proposal-template.md`
2. 填写提案内容
3. 保存为 `votes/proposal-[编号].md`
4. 创建 worktree 并开发
5. 推送代码并创建 PR

### 参与投票

1. 阅读提案文档
2. 审查代码（可选）
3. 在投票表中填写态度
4. 更新统计信息

---

## 🗳️ 投票规则快速参考

| 级别 | 法定人数 | 用户投票 | 适用场景 |
|------|---------|---------|---------|
| 一级 | ≥2 | 自愿 | 简单修复 |
| 二级 | ≥3 | 必须 | 功能新增 |
| 三级 | ≥5 | 必须 | 核心变更 |

**用户提出特批**：
- 一级、二级：用户提出可直接通过
- 三级：必须等待投票完成
- 已合并分支可通过新提案撤销

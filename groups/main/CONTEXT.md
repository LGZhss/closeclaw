# 主通道记忆

这是 CloseClaw 的主通道（self-chat）专用记忆文件。

## 当前状态

- **设置完成时间**: 2026-03-13
- **版本**: 1.0.0
- **架构**: 基于 NanoClaw

## 主通道特权

主通道是唯一可以执行管理操作的通道：

### 群组管理

- 添加新群组：`@Andy add group "群组名"`
- 移除群组：`@Andy remove group "群组名"`
- 列出所有群组：`@Andy list groups`

### 全局记忆

- 写入全局记忆：`@Andy remember [事实/偏好]`
- 仅主通道可以修改 `groups/global/CONTEXT.md`

### 任务管理

- 为任何群组创建任务
- 查看所有群组的任务
- 管理任务状态（暂停/恢复/取消）

## 使用示例

### 日常管理

```
@Andy 每天早上 8 点给我发送今日天气
@Andy 每周五下午 5 点生成周报总结
@Andy 监控 GitHub 上的新项目更新
```

### 群组管理

```
@Andy 添加群组 "家庭群" 用于 WhatsApp 家庭聊天
@Andy 添加群组 "工作群" 用于 Telegram 工作讨论
@Andy 列出所有注册的群组
```

### 全局记忆

```
@Andy 记住我偏好中文回复
@Andy 记住我不喜欢冗长的解释
@Andy 记住我正在学习 Rust 编程语言
```

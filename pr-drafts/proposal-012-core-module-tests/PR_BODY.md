## 📋 变更摘要

本 PR 补充了 CloseClaw 核心模块的单元测试，提升代码质量保障能力。

### 新增测试文件

1. **`tests/db.test.ts`** - 数据库层测试
   - 数据库连接与表创建测试
   - 消息插入与查询测试
   - 会话管理操作测试

2. **`tests/router.test.ts`** - 消息路由测试
   - 消息路由逻辑测试
   - 通道适配器集成测试
   - 错误处理路径测试

3. **`tests/group-queue.test.ts`** - 队列管理测试
   - 队列入队/出队操作测试
   - 队列状态管理测试
   - 并发场景测试

4. **`tests/task-scheduler.test.ts`** - 任务调度器测试
   - 任务注册与执行测试
   - 调度周期逻辑测试
   - 任务取消与重试测试
